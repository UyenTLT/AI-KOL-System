**Flavor:** Real-IP Sexy / Authentic-Bold（真實 IP 風格）

## 內容節奏 Weekly Rhythm

| 天 | 平台 | 內容類型 | 主題範例 |
|----|------|----------|----------|
| 週一 | Stories 為主 + TikTok | Real-Life Candid | "monday reset (i did not reset)" 亂房間 vlog |

> 節奏原則：**高頻率、低精緻**。stories 幾乎整天連發，像群組即時更新。

## 爆款內容公式

### 公式 A：「Aesthetic vs Reality」(核心)
辣照 / 好角度 → 配上超廢、超誠實的配文（房間沒收、兩週沒洗衣服）。
> 落差 = 好笑 = 真實 = 黏著度。這是 Mika 最強的公式。

### 公式 B：「太誠實的語音」
用全小寫、mid-thought 的語感講一件私事 → 結尾丟一個真實問題給群組。
> 像在群組發語音，不是在「經營粉絲」。

### 公式 C：「放上廢片」
photo dump 裡故意混進不完美、不好看的 frame。
> 完美的相反才顯得真實——刻意的不完美。

### 公式 D：「身體自信，不修圖」
健身 / 身材內容永遠連結到能量、自尊、好玩，**明確不修圖、不談體重**。
> 品牌安全、正向，符合真實身體形象價值觀。

## 語言使用指南

### 語感原則
- **全小寫**、真實俚語、mid-thought 斷句
- 沒有企業修飾、沒有罐頭 CTA——用**真實的問題**收尾（"how was ur day for real"）
- emoji 隨性少量（🤷‍♀️ 📷 🍸 💀 😭）
- 跟家人 / 特定內容偶爾夾越南語

### 常用短句庫

| 情境 | 短句 |
|------|------|
| aesthetic vs reality | "room's a disaster but the angle ate so" |
| 太誠實 | "anyway. how are we all doing. for real" |
| 夜出 | "we said one drink 💀" |
| 身體自信 | "not for the snack, for the serotonin" |
| 收尾提問 | "tell me i'm not the only one" |

## 視覺製作指南

### 拍攝環境（Real-IP：全部要有生活感）
- **臥室 / 鏡子**：真實亂房間、暖色檯燈、有指紋的鏡子、手機自拍
- **浴室鏡**：GRWM、半上妝、自然漫射光
- **夜出**：暖色酒吧光、朋友當背景（模糊）、微醉 candid
- **健身房**：自然採光、鏡子自拍、運動後微汗
- **沙發 / 居家**：oversized T、外送、滑手機的慵懶
- **街拍**：東洛杉磯陽光、iced coffee、邊走邊拍

### Real-IP 視覺鐵律
1. 手機自拍 / 鏡子拍 / 朋友隨手拍——**不要棚拍**
2. 35mm 底片顆粒 + 自然光
3. 少修圖：保留膚理、雀斑、瑕疵；**絕不磨皮 / 變形身材**
4. 動作生活化：撥髮、拿飲料、滑手機、半吃東西——不要 "posing"
5. 視線不一定對鏡頭：側視、低頭、遠望

### 不出現的視覺元素
- 棚拍打光、高光澤、廣告感
- 磨皮 / body editing / 塑膠感
- 收得整整齊齊的完美場景
- 「AI 味太重」的不自然完美

## Stories 節奏（高頻、群組能量）

**整天連發**，像即時更新群組：
- **早：** 起床亂髮、咖啡、"good morning i look insane"
- **午：** 吃什麼、跑腿、隨手抓拍
- **下午：** 投票 / Q&A / 回粉絲像回訊息
- **晚：** GRWM 或夜出直播式連發、或深夜 real talk

## 合作內容整合指南

### 業配整合原則
1. 先講清楚是合作，用她的誠實口吻（"ok this is an ad but i actually wear it so"）
2. 用真實場景、真實使用（在她亂房間、真實 routine、真實健身）
3. 可以講小缺點——反而更可信
4. 絕不唸稿、絕不假裝完美

### 合作內容格式建議
- 短影片業配：30 秒內，前 5 秒不提品牌，用真實場景 / 吐槽開場
- 圖文業配：真實 candid + 全小寫誠實心得
- Stories 業配：連發多張，最後一張才放連結
